현재 명지대학교 인문학부를 재학중인 20학번 김현빈이라고 합니다. 저의 취미는 먹고 싶은 요리를 만드는 것이고 피아노 치는 것을 좋아합니다. 요리법은 You Yube를 보면서 하는데 보통 백종원 선생님의 영상을 즐겨 봅니다. 피아노는 가요보다는 뉴에이지나 클레식을 치는 것을 좋아합니다.

사람들을 만나는 것을 좋아해서 처음 만나는 사람들과도 곧 잘 친해집니다. 만약, 지나가다 저를 만나게 되면 부담 없이 다가와 주시면 마음 속으로 되게 행복해 하고 있을겁니다! 크크
